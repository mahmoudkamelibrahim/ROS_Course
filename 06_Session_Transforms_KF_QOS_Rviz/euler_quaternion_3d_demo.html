Euler Angles — Roll · Pitch · Yaw
What are Euler Angles? Named after Leonhard Euler (1775), they describe an orientation using 3 sequential intrinsic rotations about the body's own axes (not world axes — each rotation changes the frame for the next).

R_total = R_yaw × R_pitch × R_roll

Order used here (YZX): first Yaw (world Y), then Pitch (rotated Z — now lateral), then Roll (doubly-rotated X — now longitudinal). Pitch is the middle rotation, so gimbal lock strikes at Pitch = ±90°.

How Each Rotation Works
R_roll(φ) =
[1, 0, 0 ]
[0, cos(φ), −sin(φ)]
[0, sin(φ), cos(φ)]
R_pitch(θ) =
[ cos(θ), 0, sin(θ)]
[ 0, 1, 0 ]
[−sin(θ), 0, cos(θ)]
R_yaw(ψ) =
[cos(ψ), −sin(ψ), 0]
[sin(ψ), cos(ψ), 0]
[ 0, 0, 1]
⚠ Gimbal Lock Active! At Pitch=±90°, the body's Roll axis (🔴 X) has been rotated by pitch until it points straight up — exactly where the Yaw axis (🟢 Y) already is. Both axes are now the same line in space, so Roll and Yaw sliders produce identical motion. You've permanently lost 1 DOF until pitch moves away from ±90°.
Quaternions — The Gimbal-Free Rotation
What is a Quaternion? A 4D extension of complex numbers: q = w + xi + yj + zk

For rotation, we use unit quaternions (|q|=1):
q = cos(θ/2) + sin(θ/2)(xî + yĵ + zk̂)

where (x,y,z) is the unit rotation axis and θ is the rotation angle. Invented by W.R. Hamilton in 1843.

Why Quaternions Beat Euler
✓ No Gimbal Lock — always 3 DOF
✓ Compact — only 4 numbers
✓ SLERP — smooth interpolation:
slerp(q1,q2,t) =
q1·sin((1-t)Ω)/sin(Ω) +
q2·sin(tΩ)/sin(Ω)
✓ Composition:
q_total = q2 ⊗ q1
Rotating vector v:
v' = q ⊗ v ⊗ q*
Euler ↔ Quaternion Mapping
Converting between representations: Both describe the same rotation — just different parameterizations. The mapping uses half-angle formulas and is always possible. Convention used: YZX intrinsic (Yaw→Pitch→Roll), matching the Euler tab.

Conversion Formulas (ZYX Order)
— Euler → Quaternion —
cy=cos(ψ/2), sy=sin(ψ/2)
cp=cos(θ/2), sp=sin(θ/2)
cr=cos(φ/2), sr=sin(φ/2)
w = cr·cp·cy + sr·sp·sy
x = sr·cp·cy − cr·sp·sy
y = cr·sp·cy + sr·cp·sy
z = cr·cp·sy − sr·sp·cy
— Quaternion → Euler —
φ = atan2(2(wy+xz), 1−2(y²+x²))
θ = arcsin(2(wz−xy))
ψ = atan2(2(wx+yz), 1−2(z²+x²))
